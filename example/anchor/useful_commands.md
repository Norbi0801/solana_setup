# Przydatne polecenia Anchor CLI

- `anchor keys list` – pokazuje zmapowane identyfikatory programów zapisane w `Anchor.toml`.
- `anchor idl fetch <program_id>` – pobiera IDL programu z klastra.
- `anchor localnet` – uruchamia lokalny klaster wraz z testowym faucetem.
- `anchor avm list` – wyświetla zainstalowane wersje Anchor w AVM.
