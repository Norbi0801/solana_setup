# Zapoznaj się z projektem

- `Anchor.toml` opisuje programy i klastry; nasza konfiguracja ustawia devnet i przykładowy Program ID.
- `programs/hello_anchor/src/lib.rs` zawiera logikę instrukcji `initialize` oraz `update_message`.
- `tests/hello_anchor.ts` pokazuje klienta TypeScript używającego Anchor workspace.
