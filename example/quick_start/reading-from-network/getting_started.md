# Od czego zaczniesz

- Upewnij się, że masz uruchomiony lokalny walidator (`example/cli/start_local_validator.sh`).
- Przygotuj konto testowe z airdropem SOL przy użyciu skryptu `example/cli/airdrop.sh`.
- Zainstaluj zależności klienta: `npm install` w katalogu `client` (patrz poniżej).
