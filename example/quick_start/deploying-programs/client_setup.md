# Skonfiguruj klienta

- W `client/package.json` znajdziesz skrypty `init` i `update` dla dwóch instrukcji.
- `npm install` instaluje `@solana/web3.js` i `ts-node` do uruchamiania skryptów.
- Skrypt `build_client.sh` sprawdza, czy TypeScript kompiluje się poprawnie.
