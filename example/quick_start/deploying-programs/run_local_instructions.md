# Uruchom program lokalnie

1. `example/quick_start/deploying-programs/build_program.sh` – tworzy artefakt `.so`.
2. `example/quick_start/deploying-programs/run_local.sh` – deploy do lokalnego walidatora.
3. `solana logs` – obserwuj komunikaty `Odebrana wiadomość` przy kolejnych wywołaniach klienta.
