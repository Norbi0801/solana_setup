# Ponownie wykonaj program

- Po aktualizacji konfiguracji klienta uruchom `npm run update -- "kolejna wiadomość"`, aby sprawdzić, że transakcje trafiają do nowego programu.
- Sprawdź logi `solana confirm <signature> --verbose`, aby przeanalizować status.
