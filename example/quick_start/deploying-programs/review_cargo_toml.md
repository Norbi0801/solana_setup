# Przejrzyj Cargo.toml

- Zależność `solana-program` zapewnia makra `entrypoint!` oraz funkcje logujące.
- `crate-type = ["cdylib", "lib"]` umożliwia budowę zarówno biblioteki, jak i artefaktu BPF.
- Wersję `1.18.0` możesz zsynchronizować z zainstalowanym Solana CLI (`solana --version`).
