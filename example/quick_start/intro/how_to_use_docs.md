# Jak używać dokumentacji

- Zawsze sprawdzaj wersje narzędzi (`solana --version`, `anchor --version`), bo przykłady zakładają aktualne wydania.
- Korzystaj z repozytoriów przykładowych (np. anchor-examples) jako odniesienia do pełnych implementacji.
- Oznaczaj własne eksperymenty w plikach README, by śledzić postęp.
