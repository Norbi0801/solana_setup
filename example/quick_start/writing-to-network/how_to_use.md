# Jak używać tego przewodnika

1. Wykonaj kroki sekwencyjnie: Playground → Anchor CLI → aktualizacje.
2. Po każdym kroku uruchom testy (`anchor test`), aby utrzymać pewność działania.
3. Porównuj z materiałami w `example/quick_start/start/anchor_project`, żeby zobaczyć pełny kontekst.
