# Aktualizacja programu

- Zmodyfikuj logikę w `programs/hello_anchor/src/lib.rs` (np. walidację danych).
- Skorzystaj z `anchor deploy --program-id <stary_program_id>`, aby wykonać upgrade.
- Zaznacz wynik w logach `solana logs`, żeby upewnić się, że nowa wersja została załadowana.
