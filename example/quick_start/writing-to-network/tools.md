# Z jakich narzędzi skorzystasz

## Anchor CLI
- Wdrażanie (`anchor deploy`), migracje (`anchor upgrade`), testy (`anchor test`).
- Skrypty pomocnicze: zob. `example/anchor/*.sh`.

## Solana CLI
- Transfery SOL (`solana transfer`), monitorowanie (`solana logs`).
- Airdrop i konfiguracja RPC: patrz `example/cli/*.sh`.
